# 🎙️ Alexander AI Voice

**The AI-powered voice studio by Alexander AI Integrated Solutions.**

Clone any voice. Generate speech. Dictate into any app. Give your AI agents a voice you own.

Built on open-source AI — runs locally on your machine with complete privacy.

---

## Features

- 🎤 **Voice Cloning** — Clone any voice from a few seconds of audio
- 🔊 **Text-to-Speech** — 7 TTS engines, 23 languages
- 🎙️ **Dictation** — Global hotkey to speak into any app, auto-paste
- 🤖 **Agent Voice** — Give AI assistants a voice via MCP or REST API
- 🎚️ **Audio Effects** — Pitch, reverb, delay, compression and more
- 📖 **Stories Editor** — Multi-track timeline for podcasts and narratives
- 🔒 **100% Private** — All models run locally, your voice data never leaves your machine

## Tech Stack

| Layer | Technology |
|-------|------------|
| Desktop | Tauri (Rust) |
| Frontend | React + TypeScript |
| Backend | Python FastAPI |
| TTS Engines | Qwen3-TTS, Chatterbox, Kokoro, LuxTTS, TADA |
| STT | Whisper / Whisper Turbo |
| Effects | Pedalboard (Spotify) |
| Database | SQLite |
